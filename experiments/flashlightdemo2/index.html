THIRD ITERATION!!! Now it should have support on for pretty much all browsers except IE, which, to be honest, I'm not too pressed about. Same as last time, credit to Noel Delgado's X-Ray SVG experiment which I referenced for my last two attempts, and now thank you to yoksel's clipping masking demo page for helping me find an alternative to masking HTML elements (which is somehow not supported by anyone other than Firefox).

Functionally, this is pretty much very similar to Flashlight Demo 1, with two major changes:

  1. The mask now only consists of the black circle without any white background
  2. It now uses clip-path in the CSS to clip the div, which has better support
It has pretty much all of the same benefits and functionality as last time it seems, except that clipping doesn't support using a radial gradient. My lovely soft edge :(

I do miss the visuals of the mask a bit, but other than that I'm pretty happy that it seems to work! Can totally use this!
